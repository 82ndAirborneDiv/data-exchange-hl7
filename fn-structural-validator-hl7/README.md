

### Install hl7-structural-validator

```scala
mvn install:install-file \
    -Dfile=artifacts/cdc.xlr.hl7.structurevalidator-assembly-0.3.1.jar \
    -DgroupId=cdc.xlr.structurevalidator \
    -DartifactId=cdc-xlr-hl7-structurevalidator \
    -Dversion=0.3.1 \
    -Dpackaging=jar \
    -DgeneratePom=true
```
